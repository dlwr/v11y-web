# v11n-web

Browser-based AI noise reduction recorder using WebAssembly.

## Features

- Audio recording via WebAudio API (48kHz / 16-bit / mono)
- AI noise reduction using NSNet2 model (ONNX Runtime Web)
- Real-time amplitude visualization
- Playback with Original/AI Enhanced toggle
- WAV file download

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- ONNX Runtime Web

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
